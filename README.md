## ğŸ› Common Issues

### The page looks broken!
- Make sure all files are in the right folders
- Check that `index.html` is in the main folder
- Verify the CSS and JS files are in their folders

### Colors aren't changing!
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Make sure you saved the CSS file
- Check for typos in color codes

### Form isn't working!
- The form currently shows a success message but doesn't send emails
- To actually collect emails, see the [Email Services](#-setting-up-email-collection) section

## ğŸ“§ Setting Up Email Collection

The signup form is currently a demo. To collect real emails:

### Option 1: Mailchimp (Most Popular)
1. Sign up at mailchimp.com
2. Create a list

4. Replace the form submission code in `js/script.js`

Instructions for each are in the JavaScript file!
